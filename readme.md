[📖 基础网络全科普](./基础网络/README.md)

**核心内容**：波分复用系统(DWDM/CWDM)、BGP路由协议、网络虚拟化技术(VXLAN/GRE)、VLAN技术、DHCP/DNS服务、UDP/TCP协议、三层交换与路由、ICMP/ARP协议等。

**适用场景**：为后续高级网络功能提供理论基础和概念理解，适合初学者快速补齐网络知识短板。

## 🚀 高性能数据面处理

### OVS虚拟交换机全科普
[📖 OVS全科普](./OVS全科普/readme.md)

**核心内容**：Open vSwitch(OVS)架构、vswitchd/ovsdb/datapath内核模块、流表规则配置、VLAN处理、OVS-DPDK加速方案等。

**适用场景**：理解虚拟网络交换的实现原理，掌握虚拟交换机的配置优化和性能调优。适合云网络工程师和运维人员。

### DPDK数据平面加速全科普
[📖 DPDK全科普](./DPDK全科普/readme.md)  
[📝 深入浅出DPDK笔记](./深入浅出DPDK.txt)

**核心内容**：Intel DPDK架构、EAL/Buffer Manager/PMD等核心组件、NUMA优化、内存管理(大页、缓存)、网卡多队列技术、硬件卸载(VLAN/Checksum/TSO)、性能优化策略、SPDK存储方案等。

**适用场景**：掌握用户态高性能数据包处理的核心技术和优化方法，用于网关、负载均衡器、NFV等高性能网络应用场景。

## 🌐 下一代网络协议

### IPv6网络完全指南
[📖 IPv6网络](./IPv6网络/readme.md)

**核心内容**：IPv6专业术语(RA/RS/NS/NA等)、特殊地址类型(DHCP地址、组播地址)、DHCPv6协议、邻居发现协议(ND)、RFC标准规范、管理工具等。

**适用场景**：快速理解和排查IPv6网络配置问题，支持云平台的IPv4/IPv6双栈部署。适合网络运维和故障排查人员。

### BGP路由协议及应用
[📖 BGP网络协议](./BGP网络/readme.md)

**核心内容**：BGP扩展性、策略控制、网络可靠性、安全机制(RPKI/BGPSEC)、流量工程、拓扑可见性、灵活路由决策等。

**适用场景**：理解自治系统间的路由交换，支持大规模网络中的跨域通信和流量灾备转移。广泛应用于CDN、全球加速等场景。

## 📦 云网络核心产品

### VPC虚拟私有网络
[📄 VPC产品简介](./VPC专有网络/专有网络产品简介.pdf)

**核心特性**：
- 租户隔离的虚拟网络环境
- 可自定义的IP地址范围和子网划分
- 灵活的安全策略和访问控制
- 与其他云服务的无缝集成

### NAT网关
[📄 NAT产品简介](./NAT网络/NAT网关产品简介.pdf)

**核心特性**：
- 源地址转换(SNAT)和目的地址转换(DNAT)
- 出公网流量管理
- 高可用和弹性扩展能力
- 与VPC深度集成

### 负载均衡(LB)
[📄 LB技术架构](./LB负载均衡/负载均衡技术架构.pdf)

**核心特性**：
- 四层/七层流量分发
- 多种负载均衡算法
- 会话保持和健康检查
- 支持多种后端资源

### 弹性公网IP
[📄 弹性公网IP](./弹性公网/)

**核心特性**：
- 灵活的公网地址管理
- 按需分配和回收
- 与实例快速绑定
- 支持流量限速

## 🎯 集群架构方案

### VIP虚拟IP集群架构
[📖 VIP集群详细设计](./VIP集群/readme.md)

**核心内容**：
- **IGW(内网网关)** - 统一公网网关集群，支持精准限速、多集群容灾
- **BGW(边界网关)** - 混合边界网关集群，支持自动故障转移、异构容灾

**架构特点**：
- 基于P4可编程芯片(Barefoot/Tofino)
- 集群内节点等价化设计
- BGP自动故障转移
- 多集群间相互宣告VIP实现秒级容灾

**应用场景**：公网网关、专线接入、VPN连接、跨域流量等。

## 🎓 学习路线建议

### 初学者路线
1. **基础网络** - 打牢网络基础理论
2. **IPv6网络** - 了解下一代互联网协议
3. **参考书籍** - 深度阅读TCP/IP和HTTP

### 中级工程师路线
1. **BGP网络** - 学习自治系统间的路由交换
2. **OVS全科普** - 理解虚拟网络交换原理
3. **VPC虚拟私有网络** - 掌握云网络隔离机制

### 高级架构师路线
1. **DPDK全科普** - 掌握高性能数据面处理
2. **VIP集群架构** - 应用高可用部署方案
3. **智算网络架构** - 设计面向未来的网络
4. **云网络卓越架构设计** - 阅读最佳实践白皮书
